# IoT Lab (Docker + Portainer)

A fully reproducible IoT cyber lab including:
- MQTT sensors
- Modbus wind farm
- BACnet power plant
- ThingsBoard telemetry
- Suricata + EveBox network monitoring

## Quick Start
```bash
git clone https://github.com/ethanspock/iot-lab-repo.git
cd iot-lab
sudo ./install.sh
./bootstrap.sh
```

## Portainer will need to be manually initalized with the user and password
